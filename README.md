# nmstate-generator
UI library for generating NMStateConfigs used by assisted-installer for defining host network configuration

## Project goals
- Provide easy to use tool for generating NMStateConfig resources used for providing host network configuration to the Assisted Installer discovery ISO.
- Support management of all interface types and properties available from [NMState spec](https://nmstate.io/)
- Provide a mechanism to define common configuration across all hosts
- Provide means to define configuration using UI dialogs or directly editting code in the split view.
- ...more to be added
